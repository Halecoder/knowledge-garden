卡片中的一些概念，可以聚集在一起，并打上特殊标记，也便于被搜索被引用。用字母或符号当前缀的话，字母容易造成搜索的过量匹配，符号容易不够用。最后一想，通过emoji😃来进行。

不用追求分类的完备完整收录，重要的是养成这个习惯即可，在建立新词条的时候，能有意识地用emoji符号区分开。

## 笔记的手动编码，与标题之间不加空格

| emoji | 编码 | 含义                 |
| ----- | ---- | -------------------- |
| ⚙️    | 齿轮 | app或脚本            |
| ✨    | 闪亮 | 金句                 |
| 🔗    | 链接 | 网站                 |
| 🔡    | 英文 | 英文缩写的方法或模型 |
| 🧑    | 人人 | 人物                 |
| 🪞    | 镜子 | 值得反思的俗语       |
| 🐤    | 鸡   | 网络用语 - [[小鸡词典]]                     |
